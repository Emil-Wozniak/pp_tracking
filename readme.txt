Konsolowy program do śledzenia przesyłek Poczty Polskiej

Wersja alfa


INSTALACJA:
$ sudo pip install docopt termcolor suds
$ git clone https://github.com/artekw/pp_tracking
$ chmod +x pp_tracking.py

UŻYCIE:
$ pp_tracking.py <numer przesyłki>
